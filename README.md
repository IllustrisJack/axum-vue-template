# Axum + Vue.js 3 + Vite Template
## Prerequisites:
- [Rust](https://www.rust-lang.org/)
- [pnpm](https://pnpm.io/)
## Development
Serve via ```pnpm dev``` in /frontend/
## Build
Build via ```pnpm build``` in /frontend/
